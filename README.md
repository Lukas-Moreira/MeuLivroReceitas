# MeuLivroReceitas

MeuLivroReceitas é um projeto open source desenvolvido em .NET 7, que desenvolvi ao fazer o  curso completo de desenvolvimento backend com boas práticas e padrões modernos. 

Este projeto reflete os conhecimentos adquiridos durante o curso, incluindo conceitos de Domain-Driven Design (DDD), SOLID, testes automatizados, autenticação com JWT e OAuth2, entre outros.

## Estrutura do Projeto
- `src/Backend/MyRecipeBook.Domain`         — modelagem de domínio (entidades, agregados).
- `src/Backend/MyRecipeBook.Application`    — casos de uso e lógica de aplicação.
- `src/Backend/MyRecipeBook.Infrastructure` — persistência, Unit of Work, repositórios.
- `src/Backend/MyRecipeBook.API`            — endpoints, autenticação e integração com a aplicação.
